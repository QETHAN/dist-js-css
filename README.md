dist-js-css
===========

minify js, css for dist

* options for dist


```  
{
  "domain": {
    "dev": "xxx",
    "test": "xxx",
    "prod": "xxx",
    "use": "dev"
  },
  "src": {
    "js": "./js",
    "css": "./css"
  },
  "dest": {
    "js": "./dist/js",
    "css": "./dist/css"
  }

}
```
* method for use

```
 file dirtory:
 
 /
  | /dist-js-css
  | /package.json
	
node dist-js-css.js

```
